# Governance

PR Maven CLI is founded by Will-thom.

The project is open to contributors and future maintainers. Maintainer rights should be granted based on sustained, high-quality contributions and respectful collaboration.

The current principal maintainer is Will-thom, GitHub: [@Will-thom](https://github.com/Will-thom).

Maintainer details live in [MAINTAINERS.md](MAINTAINERS.md).

## Founder Responsibilities

- Define product direction.
- Protect project scope.
- Keep the roadmap contributor-friendly.
- Review maintainer nominations.
- Preserve production reliability.

## Maintainer Responsibilities

- Review pull requests.
- Keep issues clear and scoped.
- Improve contributor onboarding.
- Protect stable CLI and JSON contracts.
- Prefer tested, incremental changes.
- Merge only changes that match the roadmap stage, pass the required CI gate, and preserve the project scope.

## Contributor Path

1. Open focused PRs with tests.
2. Participate in issue discussion.
3. Help improve fixtures, docs, and parser coverage.
4. Earn maintainer trust through consistent reviewable work.

## Decision Making

The project uses founder-led consensus.

For now, Will-thom has final decision authority on scope, roadmap, release timing, and maintainer access.
